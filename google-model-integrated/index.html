Google Model Suite Now Available on Modellix — Nano Banana, Imagen 4.0 & Veo Up to 4K

April 9, 2026
Modellix Google New Models AI
Modellix Team Official
Summary

Modellix now supports Google's complete AI media model family — Nano Banana for fast image generation and editing, Imagen 4.0 for photorealistic output, and the full Veo 2/3/3.1 lineup for video generation up to 4K. Select Veo models are priced below market.

We’re excited to announce that Google’s full AI media model suite is now available on Modellix — accessible through the same unified API you already use for every other model on the platform.

This is the largest single-provider integration we’ve shipped to date: 21 models spanning image generation, image editing, and video generation — from fast, cost-effective options to flagship 4K-capable models.

What’s New

Nano Banana — Fast, Versatile Image Generation & Editing

Google’s Nano Banana family covers both generation and editing workflows:

  • Nano Banana — Text-to-image with 10 aspect ratios. Fast and cost-effective.
  • Nano Banana Pro — Higher quality output with resolutions up to 4K.
  • Nano Banana 2 — Multimodal model supporting both text-to-image and image-to-image, with 14 aspect ratios and resolutions from 512 to 4K.
  • Nano Banana Edit / Pro Edit / 2 Edit — Image editing variants that transform existing images based on prompt instructions, with Pro Edit offering superior detail preservation and up to 4K output.

Imagen 4.0 — Photorealistic Image Generation

Google’s latest flagship image model, available in three tiers:

  • Imagen 4.0 — High-quality photorealistic images with batch generation (up to 4 images), person generation control, and up to 2K resolution.
  • Imagen 4.0 Ultra — Maximum detail and photorealism for the highest quality output.
  • Imagen 4.0 Fast — Speed-optimized variant with the same feature set for rapid iteration.

Veo 3.1 — Flagship Video Generation Up to 4K

Google’s most capable video generation models:

  • Veo 3.1 T2V / I2V — Text-to-video and image-to-video at resolutions up to 4K. Supports up to 3 reference images for style and character consistency. Duration: 4/6/8 seconds.
  • Veo 3.1 Fast T2V / I2V — Faster variants with the same 4K capability and reference image support.

Veo 3 — Stable 1080p Video Generation

  • Veo 3 T2V / I2V — Stable text-to-video and image-to-video at up to 1080p with person generation control. Duration: 4/6/8 seconds.
  • Veo 3 Fast T2V / I2V — Faster variants with the same 1080p output quality.

Veo 2 — Classic, Reliable Video Generation

  • Veo 2 T2V / I2V — Google’s proven video generation models with flexible person generation policies (allow all, adult only, or disallow). Duration: 5/6/8 seconds.

Competitive Pricing on Veo

We’ve worked to bring Google’s video models to Modellix at prices that are genuinely competitive — and in several cases, below what other platforms charge for the same Veo models.

If you’re already using Veo through another provider, it’s worth comparing. Check our pricing page for exact per-second rates across all Veo 2, Veo 3, and Veo 3.1 variants.

Same API, Zero Integration Work

If you’re already using Modellix, calling Google models requires no code changes beyond swapping the model_slug. The unified API handles authentication, parameter normalization, and result delivery identically across all providers.
